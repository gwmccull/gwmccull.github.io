Useful git commands:
`git init`

`git add .`

`git commit -am "commit message"`

`git fetch --all`

`git pull`

`git pull origin <branch>`

`git push origin <branch>`

`git branch -l`

`git branch -la`

`git checkout <branch>`

`git checkout -b <branch>`

`git branch -d <branch>`

`git merge <branch>`

`git log`

`git reflog`

`git diff <branch>`
